# mistkereporterr

Mistake Notebook — a small single-file HTML app to log and review DSA mistakes.

Features
- Log problems, what went wrong, and the correct approach.
- Filter by topic and status (needs revision / mastered).
- Export to JSON.

Usage
1. Open mistake-notebook.html in your browser.
2. Add an entry and use the UI to manage your mistakes.

Storage
- The app stores entries in browser localStorage under the key dsa-mistakes.

Files
- mistake-notebook.html: the full app (single file).
